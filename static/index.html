product Demo REST API Service

GET /products to see all the products.

GET /products?category=cellphones to see all of the products in the 'cellphones' category.

GET /products/{id} retrieves a product from the DB using an ID.

PUT /products/{id} updates a product using an ID.

DELETE /products/{id} removes a product from the DB using an ID.

POST /products creates an new product.

PUT /products/{id}/discontinue discontinues a product.

GET /products?min-price=10&max-price=20 or GET /products?price=100 queries the products by prices.

GET /products?category=category_name queries the products by description

Example Data:

    {
        "name": "iPhone 7",
        "category": "cellphones"
    }
